Mis Servicios

Diseño Web

En este curso de diseño web aprenderás a utilizar herramientas como Adobe Photoshop, Illustrator y Sketch para crear diseños atractivos y funcionales para sitios web, así como a utilizar tecnologías de front-end como HTML, CSS y JavaScript para implementar tus diseños y crear una experiencia de usuario intuitiva.

Desarrollo Web

En este curso de desarrollo web aprenderás a utilizar lenguajes como HTML, CSS y JavaScript para crear páginas web interactivas y responsivas, así como a utilizar frameworks populares como Bootstrap y jQuery para mejorar la eficiencia de tu trabajo.

Base de Datos

En este curso de base de datos aprenderás a diseñar, crear y administrar bases de datos relacionales utilizando herramientas populares como MySQL y Oracle, así como a utilizar lenguajes de consulta como SQL para realizar operaciones avanzadas de filtrado, búsqueda y actualización de datos.
